## 1. 개요

**마크다운(Markdown)** 은 일반 텍스트 기반의 **경량 마크업 언어**이다.

### 장점

- 문법이 쉽고 간결하다!
- 관리가 쉽다!
- 지원 가능한 플랫폼과 프로그램이 다양하다!

### 단점

- 표준이 없다!
- 모든 HTML 마크업을 대신하지 못한다!

## 10. 표

### 왼쪽 정렬(기본값)

**position 속성**

| 값       | 의미              | 기본값 |
| -------- | ----------------- | ------ |
| static   | 기준 없음         | O      |
| relative | 요소 자신         | X      |
| absolute | 위치 상 부모 요소 | X      |
| fixed    | 뷰포트            | X      |

### 가운데 정렬

**position 속성**

|    값    |       의미        | 기본값 |

|  fixed   |      뷰포트       |   X    |
